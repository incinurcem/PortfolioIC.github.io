Project 3: Comparison of Regression Methods with tabular data

Jan 10, 2023

        It is a project based on the analysis of table data taken from the website with twelve different regressors with a detailed exploratory data analysis stage. As a result of the analysis, it is aimed to compare R2 Square Score.


    What's been done

  • Using the read_html feature of the Pandas library, a table containing data on world population was retrieved from a website.
  • The imported table data is arranged and assigned to a dataframe.
  • New columns were added to the dataframe by extracting properties from other columns, namely population status and development status.
  • Exploratory data analysis stages were applied on the data in detail.
  • For modeling, the data is divided into train and test with a 20% test rate.
  • Analysis was performed with 12 different regression models. These models are;
    • Linear Regressor
    • Polynomial Regressor
    • Ridge Regressor
    • Lasso Regressor
    • ElasticNet Regressor
    • Decision Tree Regressor
    • Random Forest Regressor
    • Adaptive Boosting Regressor
    • Gradient Boosting Regressor
    • Extreme Gradient Boosting Regressor
    • Support Vector Regressor
    • Stacking Regressor


    Conclusion and Interpretation

        As a result of the analysis with twelve different classifiers, the classifier with the highest R2 Square Score was Gradient Boosting Regressor with 76.8%.

        R2 Square Score is the explanation of the total change in the dependent variable (target) with the change in the independent variables (features). For this reason, according to the analysis and the dataset used, the Gradient Boosting Regressor is the regressor that best explains the change of the dependent variable, which is Yearly Change, according to the change of the values of the other columns.

        The RMSE value is the square root of the mean difference between the true value and the predicted value. With a 54% RMSE score, this difference was at least achieved with Gradient Boosting Regressor.
