Project 2: Comparison of Visual Classification Techniques

Jan 10, 2023

        It is a project based on the analysis of photos taken from Google Images, divided into four different categories, with five different classifiers. As a result of the analysis, it was aimed to make an Accuracy Score comparison.

    What's been done

  • Using the Selenium library, photos were found in the desired tag from Google Images and downloaded to the desired path.
  • A dataset consisting of a total of 2000 photographs, 500 for each category, was obtained.
  • The names of the downloaded photos have been changed sequentially according to the category for ease of editing.
  • For the edited photos, exploratory data analysis was carried out at 150x150 image size.
  • The data is separated manually according to 20% test rate and assigned to separate dataframes.
  • For the analysis of the data, 5 different classifiers were used during the modeling phase. These are;
    • Linear Support Vector Classifier
    • K-Nearest Neighbors Classifier
    • Gaussian Naive Bayes Classifier
    • Random Forest Classifier
    • Decision Tree Classifier

    Conclusion and Interpretation

        As a result, Random Forest Classifier obtained the highest accuracy rate of 66.7% among the five different classifiers used. This result can be optimized by increasing the size of the dataset.

        Accuracy score is the rate at which our model provides accurate predictions. For this reason, according to the analysis and the dataset used, Random Forest Classifier predicts better than other classifiers which of the four different categories a photograph randomly given to our model belongs to.
